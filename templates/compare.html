Key Findings
Understanding the results

🤖 ML Attacks (Gradient-Based)

  • FGSM — Fast but less powerful; single gradient step
  • PGD — Strongest white-box attack; iterative optimization
  • Both require gradient access (white-box setting)
  • Can be blocked by gradient masking (but EC attacks bypass this)

🧬 EC Attacks (Gradient-Free)

  • Genetic Algorithm — Population-based search, gradient-free
  • Differential Evolution — Continuous optimization in reduced space
  • Black-box attacks — only need model outputs
  • Can bypass gradient-masking defenses (distillation)

🛡️ Defense Effectiveness

  • Adversarial Training — Generally most robust defense
  • Input Transformation — Easy to apply but adaptive attacks bypass it
  • Detection Network — Can catch known attacks but misses novel ones
  • Distillation — Smooths gradients but EC attacks bypass this

⚠️ Key Takeaway

No single defense is sufficient. Gradient-masking defenses (distillation, input transformation) can be bypassed by gradient-free EC attacks that don't rely on gradient information.

Adversarial Training provides the most consistent defense across both ML and EC attack types, but comes at the cost of higher training time and slightly reduced clean accuracy.
